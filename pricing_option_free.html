1. Whats inside the Free package?

The Bitcoin Development Kit for free comes equipped with everything you need to get you started with integrating into the Bitcoin network to the fullest extent. Basically the library source code and the AGPL 3.0 license. You'll be able to access the Documentation, API list, and Wiki files online(or you can download them for offline use that is fine as well).


2. What are the benefits to purchase the BDK library?

No matter how much you pay us we will always provide you with the same source code library, So why pay? The benefits of paying is the more available resources you receive with your source-code to help you better understand how to properly utilize the kit.

Things like fully-fledge text & video combination of tutorials to help with all types of learners!

Not to mention example scripts that range from Simplistic 5 lines of code example scripts to complex database integration scripts.

Our tutorials cover every single function that is available for useage from this kit, but please note that this kit has functions being added all the time and as time goes on more tutorials will be available so you might be sitting here wondering... "Will I have to pay for the newly available tutorials even after I have purchased and received my kit?". Nope! You get updates for life, you will be notifed by email when new download able tutorials are available for you to view.

Who could have thunk Bitcoins could be so painless?
