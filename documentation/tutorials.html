2. How do I integrate the BDK library into my code?

* Download your BDK library for free(or purchase) your library, reffer to the pricing page for download options.

* Once the library/kit downloaded, unzip the compressed folder into your development working directory (Anywhere your PHP code can easily access the BDK folder library)

* Now that the BDK library is accessable by your code we need to link the BDK library to your code to make it aware of the new functions it has. So now we must include the bdk library like the following code:

<?php
	//Include the BDKp library into our code
	require_once("/path/to/library/folder/BDKp_library.php");
?>

* Now we need to configure our library, open up your config.php file and edit it to your servers' appropriate settings.

The default config file looks like the following (Please note we have removed all comment-headers for brevity, we have left the one-liners though, Your welcome!):

<?php
	$bdk_settings["hash_type"] = "sha256"; //What should the hash() function use?
	$bdk_settings["coin_authentication_timeout"] = 1200; //How much time (in seconds) how long a user has to authenticate their Bitcoin identity before a signature is considered expired

	//Define some Bitcoin client configuration settings
	$btcclient["https"]	= "http"; //HTTPS is recommended....
	$btcclient["host"]	= "127.0.0.1"; //Just the domainname don't put Http:// or https:// that is already taken care of.
	$btcclient["user"]	= "username";
	$btcclient["pass"]	= "password";
	$btcclient["port"]	= "4367";

	//Define Integrity checks (checksum details)
	$bdk_integrity_check = 'TypeALongRandomStringHere'; //Generate a random string that is atleast 4096 characters long, Random number here: http://textmechanic.com/Random-String-Generator.html

?>

* Great Job! Now we need to make use of our code! On to the first real tutorial...
